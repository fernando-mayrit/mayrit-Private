# Frontend de Mayrit (React + TypeScript)

Pendiente de inicializar (requiere Node.js LTS instalado). Cuando lo esté, se creará con Vite:

```
npm create vite@latest . -- --template react-ts
npm install
npm run dev
```

De momento es un marcador de posición: en la fase actual (modelo de datos + backend) no se necesita Node.
